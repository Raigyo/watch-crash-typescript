## How to compile?

`tsc index`

`tsc -- watch index`

`tsc`

`tsc -- watch`

## How to create a _tsconfig.json_?

`tsc --init`

Exemple:

```json
   "outDir": "./dist",
   "rootDir": "./src",
```
